Adapt 2 Me Grid System Readme

This grid system was drew from the inspiration created by Dreamweaver's Mobile First Responsive Grid Approach, the 960 Grid System, and Twitter Bootstrap.

- Rows -

Rows define the different layers of the site by areas grouping the top level areas.

- Grid_* -

Define the main sectors of the site with a grid.

- Nest_* -

The nest areas are sub divisions of the initial grid sectors.
